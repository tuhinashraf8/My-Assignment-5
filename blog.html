what is the Document object Model (DOM)

Ans. (DOM) হলো ‍Document Objects Model এর সংক্ষিপ্ত রূপ যদি ভেঙে ভেঙে বলি তাহলে এভাবে বলা যায় HTMLপেজ হল একটি Document এবং HTML এর ভিতরে থাকা elements গুলো হলো Objects আর এই Objects গুলোর মধ্যে যে গাছের লতা পাতার ন্যয় ছড়িয়ে আছে সেটাকে বলে Mode .

সোজা বাংলায় বলা যেতে পারে (DOM) হলো একটি গাছের স্ট্রাকচারে মত শেকর থেকে শুরু করে অনেক শাখা প্রশাখা ছড়িয়েছে। তবে এটার শেকর টা উপরের দিকে।
(DOM) এর মাধ্যমে আমরা একদম শেকর থেকে শুরু করে গাছের প্রতিটি পাতায় পাতায় আলাদা আলাদা ভাবে যেতে পারি এবং সেটায় একটি নির্দিষ্ট আইডি ব্যবহার করে জাভাস্ক্রিপ্ট এর মাধ্যমে তার মাঝে বিভিন্ন কিছু করতে পারি। যেমন জাভাস্ক্রিপ্ট ব্যবহার করে ওয়েব পেজের বিভিন্ন এলিমেন্টকে পরিবর্তন, যোগ বা মুছে ফেলতে পারি। উদাহরণস্বরূপ, আপনি একটি বাটনে ক্লিক করলে একটি প্যারাগ্রাফের টেক্সট পরিবর্তন করতে পারেন। সেটার মান বের করতে পারেন , style পরিবর্তন করতে পারেন ইত্যাদী।
সুতরাং ‍আমরা বলতে পাড়ি Document Object Model (DOM) হলো একটি প্রোগ্রামিং ইন্টারফেস যা HTML এবং XML ডকুমেন্টের জন্য ব্যবহৃত হয়। এটি একটি ওয়েব পেজের কন্টেন্ট, স্ট্রাকচার এবং স্টাইলকে ডাইনামিক্যালি অ্যাক্সেস এবং আপডেট করতে সাহায্য করে।

Ans. We can select Element From DOM in 5 common wey

sirial number way to get element example

2 By Class Name: const element = document.getElementById("elementId");

What is event delegation in the context of the dom, and Why is it useful?

Ans. Event হলো কোন একটা action যেটার মাধ্যমে ব্রাউজারের বিভিন্ন কার্যসম্পাদন করা যায় প্রায় ৮০ ৯০ টার মতো Event আছে তবে এর মধ্যে সবচাইতে কমন Event হলো ক্লিক ডমের মধ্যে addEventListener মাধ্যমে বিভিন্য ধরনের Event যোগ করা যায়
Event delegation একটি Action যেটার মাধ্যমে DOM এর উচ্চ স্তরে Event হ্যান্ডেল করা যায়। এবং এর মাধ্যমে প্রতিটি এলিমেন্টে Event Listener যোগ না করে একটি parent এলিমেন্টে Event Listener যোগ করে ইভেন্ট বাবলিং এর মাধ্যমে তার Child গুলোকে নিয়ন্ত্রন করা যায় ।

এইটা ইউসফুল হবার কারণ
1= যখন অনেক Element থাকে তখন এটির মাধ্যমে Event Listener সংখ্যা কমিয়ে আনা যায় এবং পারফরম্যান্সকে উন্নত করে।
2= ডাইনামিক কন্টেন্টে ইভেন যোগ করে বাকি গুলোকে হেন্ডেল করা যায় । যার ফলে নতুন যোগ করা এলিমেন্টগুলির জন্য আলাদা Event Listener যোগ করার প্রয়োজন হয় না।
3=কোডকে সহজ এবং পরিষ্কার রাখে, কারণ ইভেন্টগুলি কেন্দ্রীয়ভাবে পরিচালিত হয় যার ফলে অনেক কোড লোখা লাগে না।

How do you Manipulate an element's attributes and style using the dom

Ans. we can use three methods in tow
The three methods is 1: set 2: get 3: remove
And the tow is 1: Attribute 2: style property
Example

sirial number way to get element example
1 Setting an Attribute: const element = document.getElementById("myElement"); const className = element.getAttribute("class"); console.log(className);
2 Getting an Attribute: const element = document.getElementById("myElement"); const className = element.getAttribute("class"); console.log(className);
3 Removing an Attribute: const element = document.getElementById("myElement"); element.removeAttribute("class");
4 Setting a Style: const element = document.getElementById("myElement"); element.style.color = "blue"; element.style.backgroundColor = "lightgray";
5 Getting a Style: const element = document.getElementById("myElement"); const color = element.style.color; console.log(color);
6 Removing a Style: const element = document.getElementById("myElement"); element.style.color = "";
